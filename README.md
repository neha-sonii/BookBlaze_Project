## Create .env file

```
# server
PORT=YOUR_PORT_NUMBER

# PostgreSQL Database
PG_HOST=HOST_NAME
PG_PORT=DATABASE_PORT
PG_USER=your_username
PG_PASSWORD=your_password
PG_DATABASE=DATABASE_NAME

# JWT Secret
JWT_SECRET=your_jwt_secret

# External APIs
GOOGLE_BOOKS_API_KEY=your_api_key
```
